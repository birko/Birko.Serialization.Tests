# Birko.Serialization.Tests

Unit tests for Birko.Serialization, Birko.Serialization.Newtonsoft, Birko.Serialization.MessagePack, and Birko.Serialization.Protobuf.

## Test Framework

- **xUnit** 2.9.3
- **FluentAssertions** 7.0.0
- **.NET 10.0**

## Running Tests

```bash
dotnet test Birko.Serialization.Tests.csproj
```
